# Getting Started

# API Workflow App

A Next.js application scaffold.

## Quick Start

```bash
pnpm install
pnpm dev
```

## Stack
- **Framework**: Next.js (App Router)
- **Styling**: tailwind
- **State**: tanstack-query
- **Auth**: None (add your own)

## Project Structure

```
src/app/
  layout.tsx        # Root layout with providers
  page.tsx          # Home page
  globals.css       # Global styles
src/components/
  providers.tsx     # Client-side providers
```
